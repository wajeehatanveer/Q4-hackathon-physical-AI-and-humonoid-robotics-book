# Understanding URDF (Unified Robot Description Format) for humanoids

This chapter delves into the Unified Robot Description Format (URDF) and its importance in modeling humanoid robots.

## URDF Basics

Understand the XML-based syntax of URDF to define the kinematic and dynamic properties of a robot.

## Modeling Humanoids

Learn how to use URDF to accurately model complex humanoid structures, including joints, links, and sensors.

## URDF in Simulation and Visualization

Explore how URDF models are used in robot simulators like Gazebo and visualization tools like RViz.
